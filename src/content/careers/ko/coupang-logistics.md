### 주요 성과

- **로켓설치 Open API 플랫폼 구축:** 로켓설치 파트너사와의 배송/반품 시스템 연동을 위한 Open API 설계 및 구축. **삼성전자, LG전자, SK Magic, Winion** 등 대형 파트너사와 실제 연동
- **'실시간 배송/반품 현황 대시보드' 개발:** 전국 물류 거점 단위(쿠팡맨, 캠프, 에어리어, 리전, 전국)의 실시간 배송/반품 현황 대시보드 개발 (Elasticsearch 활용)
- **운영 지표 리포팅 자동화:** 6종의 핵심 운영 지표(배송/반품 지연 및 물류 출고 현황 등)에 대한 리포팅 자동화 시스템 개발
- **3PL 택배사(롯데택배) 연동:** 롯데택배와 3PL 연동(배송/반품 트래킹, 정산)

### 로켓 설치 프로젝트

- 쿠팡이 추후 추가 개발 없이 제조사들을 추가할 수 있도록, 제조사들에게 제공할 Partner API 설계
- 제조사별&우편번호별 배송 휴일 달력을 제작하고, 휴일 정보를 제조사들과 동기화
- **SK Magic**을 시작으로, **Winion, LG전자** 등 이후 연동하는 모든 제조사들과의 연동에 담당자로 참여
- 위 4개의 외부 제조사들의 개발자와 직접 커뮤니케이션을 했고, 함께 스펙 결정, 개발 및 테스트 진행
- 오픈 시작한지 한 달 만에 수억의 매출 달성

### 배송/반품 관련 통계 대시보드 제작

- 쿠팡친구/캠프/에어리어/리전/전국으로 모든 단위를 커버하는 '실시간 배송현황 대시보드' 개발
- 일간 약 백만 건에 달하는 송장들의 실시간 완료 여부, 쿠팡친구 시간당 배송 완료 수, 유휴시간, 배송 효율, 최종 배송 완료 소요시간 등 자세한 데이터 처리
- Elastic Search를 사용하여 데이터를 쌓고, 단위별로 aggregation하여 데이터 추출

### 롯데택배 연동

- 쿠팡의 3PL 택배사에 롯데택배 합류
- 배송 트래킹, 반품 트래킹, 정산 담당
- 정산은 택배사의 정산 데이터를 HBase에 쌓고, 쿠팡의 데이터와 비교 및 통계를 계산하는 방식으로 진행
